## Getting Started

### Prerequisites

- npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/mandar00/TasksApp.git
   ```
2. Move into tasks folder
   ```sh
    cd TasksApp
   ```
2. Install NPM packages
   ```sh
   npm install
   ```

### Start

- Run project locally
  ```sh
  npm run dev
  ```
